# MD 转图片服务

一款本地轻量 Markdown 转 PNG 图片工具，开箱即用，无需复杂配置，适合快速将文档渲染为高清长图。

## ✨ 功能特点

- 支持粘贴文本、拖拽、上传 MD 文件

- 完整兼容：表格、代码块、列表、引用等语法

- 一键生成无损 PNG 长图，自动保存至 output 文件夹

- 自带明暗双主题，自定义图片宽度

- 自动识别浏览器，启动自动打开网页

- 内置版本检测，提示更新

## 🚀 快速使用（推荐）

### 方式一：EXE 直接运行（零基础）

1. 前往 [Releases](https://github.com/ZENGZENGQH/md-to-image-service/releases/latest) 下载最新压缩包

2. 解压，双击 `exe` 程序

3. 浏览器自动弹出，直接使用

### 方式二：源码运行

```bash
# 安装依赖
npm install

# 启动服务
npm start
```

## 💻 环境要求

- 操作系统：Windows

- 需安装 Chrome 或 Edge 浏览器

- 源码运行：Node\.js \&gt;= 16

## 📁 目录说明

- `output`：自动保存生成的图片
